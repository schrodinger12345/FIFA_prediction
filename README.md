 # FIFA Prediction

This project focuses on predicting FIFA/football match outcomes using historical data and machine learning techniques such as Random Forest and XGBoost.

## Overview

The repository is intended for data preparation, feature engineering, model training, and evaluation for football prediction tasks.

## Features

- Data cleaning and preprocessing
- Feature engineering for football data
- Model training and evaluation
- Prediction generation for new inputs

## Requirements

- Python 3.x
- pandas
- numpy
- scikit-learn
- xgboost
- joblib

## Installation

```bash
git clone <repository-url>
cd FIFA_prediction
pip install -r requirements.txt
```

## Usage

Hop directly onto the last cell and change the arguements of the custom prediction function. Once you've entered your own arguements run the cell

## Typical Workflow

1. Load the dataset
2. Clean and preprocess the data
3. Create useful features
4. Train the model
5. Evaluate performance
6. Make predictions
